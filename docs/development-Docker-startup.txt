# Development Environment â€” Docker Startup

This file documents the commands for starting and using the local development stack.

## Start the dev environment

  .\dev.ps1 up -d --build

  Or without dev.ps1:

  docker compose -f docker-compose.yml -f docker-compose.dev.yml up -d --build

## First run: Initialize the database

After starting, run migrations and seed:

  .\dev.ps1 exec api pnpm db:deploy
  .\dev.ps1 exec api pnpm db:seed

  Or without dev.ps1:

  docker compose -f docker-compose.yml -f docker-compose.dev.yml exec api pnpm db:deploy
  docker compose -f docker-compose.yml -f docker-compose.dev.yml exec api pnpm db:seed
